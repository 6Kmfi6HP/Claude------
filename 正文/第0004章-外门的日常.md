# 第4章 外门的日常

三月十三，清晨。

天玄宗外门的钟声准时响起，沉闷而悠长。

陈默从被窝里爬起来，揉了揉眼睛。窗外天才蒙蒙亮，晨雾还没散，空气里带着草木的湿气。

他穿上那件洗得发白的灰色道袍，出门往练功场走。

外门的早课是规矩，没人敢逃——管事每天都会点名，缺席三次以上，当月灵石配给直接清零。

陈默不想跟灵石过不去，虽然他那点配给也少得可怜。

练功场上已经聚集了上百号外门弟子。

有的在打拳热身，有的在盘膝吐纳，有的三三两两聚在一起闲聊。

陈默找了个靠后的位置站好，打了个哈欠。

“都站好了！”

一个中年男人的声音从前方传来。

外门管事，姓王，筑基初期修为，在外门干了十几年，脸永远板着，像谁都欠他灵石似的。

“今日早课照常。第一项，点名。”

王管事从袖子里掏出一卷名册，一个一个念。

念到陈默的时候，他顿了一下。

“陈默。”

“到。”

王管事抬起头，目光越过人群，落在陈默身上。

“陈默……练气三层，入门三年。三年了，一点进步都没有？”

陈默没说话。

“我在问你话。”

“回管事，是。”

王管事哼了一声，把名册往下一翻。

“三年练气三层，你是我在天玄宗见过的最差劲的弟子之一。宗门养着你，是让你来吃白饭的？”

全场的目光齐刷刷聚到陈默身上。

有人偷笑，有人摇头，有人面无表情。

陈默面不改色：“回管事，弟子资质愚钝，给宗门丢脸了。”

王管事盯着他看了两秒，似乎想从他脸上找到一丝羞愧或慌张，但什么也没找到。

他皱了皱眉，没再说什么，继续往下点名。

站在陈默旁边的瘦高个儿用胳膊肘捅了捅他，压低声音：“你也太能忍了，他那么说你你都不还嘴？”

“还嘴干嘛？他又没说错。”

“你就不生气？”

“生气有用吗？生气能让我突破？”

瘦高个儿一时语塞。

“所以啊，”陈默双手插袖，“省点力气。”

瘦高个儿叫张小满，也是外门弟子，练气四层，入门比陈默早一年。性格咋咋呼呼，是陈默在宗门里为数不多能说上话的人。

“对了，”张小满凑过来，“你听说了没？三个月后的宗门大比，前三名的奖励翻倍了！”

“哦。”

“筑基丹、灵石五百、功法阁三层通行证！三层啊！内门弟子才能去的地方！”

“哦。”

“你就不能有点反应吗？”

“筑基丹我又用不上。我现在练气三层，吃筑基丹是暴殄天物。”

“那你可以先留着啊！等你筑基了再用！”

“我离筑基还差六个境界，先不急。”

张小满张了张嘴，发现好像确实没法反驳，只好闭嘴。

早课在管事的训话中结束。

散场的时候，弟子们三三两两往外走，话题自然而然地转到了宗门大比上。

“听说这次内门也会派人参加，往年都是外门自己玩。”

“真的假的？那我们还比什么？”

“就是啊，内门弟子最差也是筑基期，根本打不过。”

“所以这次前三奖励才翻倍嘛，宗门也知道外门没戏。”

“也不一定——我听说柳鸣师兄这段时间练得特别狠，天天泡在练功场。他练气七层了，说不定能跟内门的拼一拼。”

“柳鸣？那个前年入内门的？”

“对，就是他。天赋高，又拼，今年才二十一岁。很多人都看好他拿这次大比的头名。”

陈默听到这个名字，脚步微微顿了一下。

柳鸣。

他记得这个人。

三年前他入宗的那天，就是柳鸣在迎新会上当着所有人的面说他“这种资质也配来天玄宗”。那句话让他在外门彻底成了透明人。

不过那也是三年前的事了。

陈默摇了摇头，把这些念头甩开，继续往食堂方向走。

他经过宗门广场的时候，看到大比报名处已经搭起来了。

一张红木桌，后面坐着两个内门弟子在登记。桌上摆着一块玉牌，上面用灵力刻着几个大字：

“宗门大比·报名处”

不少弟子围在桌前，有报名的，有打听规则的，有单纯看热闹的。

陈默本来想绕过去，但人实在太多，他只能从人群边上挤过去。

就在他快要穿过人群的时候，一只手忽然从旁边伸出来，按住了他的肩膀。

“哟，这不是陈默吗？”

声音不大，但周围好几个人的目光都被吸引了过来。

陈默转过头。

一个身材挺拔的年轻人站在他面前，穿着内门弟子的青色道袍，腰间挂着一柄长剑，嘴角挂着似笑非笑的表情。

练气七层。

陈默认出了这张脸——虽然比三年前成熟了一些，但那副居高临下的神情一点没变。

柳鸣。

“好久不见啊。”柳鸣上下打量了他一番，“三年了，练气三层？”

陈默没接话。

“我刚才听王管事点名的时候还以为是同名，没想到真是你。”柳鸣语气很随意，但声音不大不小，刚好让周围的人都听得见，“三年一点进步都没有，你当初是怎么通过入门考核的？”

旁边几个外门弟子互相交换了一下眼神，悄悄往后退了半步，给两人留出空间。

陈默平静地看着他：“入门考核是一对一实战，我打赢了。”

“哦？”柳鸣挑了挑眉，“打赢了？我记得你那场的对手是——一个练气二层的杂役吧？你练气三层打练气二层，赢了很光荣？”

“赢了就是赢了。入门考核没规定不能打比自己弱的。”

柳鸣被这句话噎了一下，随即笑了：“三年前你没这么会说话。”

“三年前你也没这么爱管闲事。”

周围有人倒吸一口凉气。

柳鸣的笑容僵了一瞬。

他看着陈默，目光变得有些锐利：“听说你也打算参加大比？”

陈默想说“我没打算”，但话到嘴边，他忽然想到——如果他承认了，系统肯定会立刻弹出任务进度提醒，那也太丢人了。

所以他换了个说法：“还没想好。”

“还没想好？”柳鸣笑了一声，“也对，练气三层参加大比，第一轮都过不去。我要是你，我也想不好。”

陈默没生气。

但他感觉到脑海里有什么东西动了一下。

【检测到宿主受到外部压力刺激。系统建议：接受挑战，使用系统辅助方案进行针对性训练——】

他默默在心里对系统说了一句：闭嘴。

系统的声音消失了。

陈默抬起头，看着柳鸣的眼睛，笑了笑。

“你说得对，我确实打不过。所以报名的事，我再想想。”

然后他侧身从柳鸣身边走过去，头也不回地往食堂方向去了。

柳鸣站在原地，看着他的背影消失在人群里，脸上的笑容淡了几分。

旁边一个内门弟子凑过来：“柳师兄，你认识他？”

“算是吧。”柳鸣收回目光，“三年前入宗的时候，我看走眼了。”

“走眼？他不是废物吗？”

“是啊，废物。”柳鸣顿了顿，“但废物被我踩了三年，从来没回过头。”

他想起刚才陈默看他的眼神——平静，毫无波澜。

不是强忍怒火的那种平静，是真的不在乎。

好像他柳鸣这个人，在陈默的世界里根本不值一提。

这种眼神，比被人骂一百句废物还让人不舒服。

报名处前的喧闹还在继续。

但柳鸣已经没了兴致。他转身往练功场走去，脚步比来时快了几分。

食堂里，陈默端着碗稀饭坐在角落里。

系统的声音终于憋不住了。

【宿主，刚才那个人的挑衅行为——系统数据库中标记为“打脸预备场景”。系统建议宿主积极回应挑战，以提升在宗门内的声望值。】

“声望值有什么用？能换灵石吗？”

【虽然系统暂未开放声望值兑换功能，但较高的声望有助于宿主获取更好的修炼资源——】

“那也得等我打得过他再说。”

【宿主当前修为为练气三层，目标为练气七层。根据方案C的训练计划，三个月后宿主有望达到练气四层。届时配合系统战术辅助——】

“练气四层打练气七层？”

【仍有显著差距。】

“那你说什么？”

【但系统认为——宿主的态度正在从不合作向现实妥协转变。这是积极的信号。】

陈默喝了一口稀饭，暗自笑了笑。

这系统，居然连“态度转变”都能监测出来。

“行了行了，先吃饭，修炼的事下午再说。”

【宿主终于主动提及修炼。】

“我就是随便一说。”

【但系统已经记录了。】

陈默无奈地摇了摇头。

三月十三上午的阳光照进食堂，在灰扑扑的桌面上投下一道光斑。

平凡的一天，才刚刚开始。

【未完成任务积压：3。】

【第4章·完】
