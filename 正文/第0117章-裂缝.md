# 第117章 裂缝

他们沿着屏障走了一整天。

从正午走到傍晚，从傍晚走到暮色四合。神秘人在前面，陈默在后面，白色的屏障在他们左侧绵延不绝，像一堵没有尽头也没有缺口的墙。每一段看起来都和上一段一模一样——同样的纹理，同样的颜色，同样的高度，同样的无始无终。

没有门。没有入口。没有任何像开口的东西。

天黑之后，神秘人停了下来。

他没有说话，但陈默从他的沉默中读出了他不想说出口的东西——他找不到入口。他沿着这道屏障走了很多年，从东走到西，从南走到北，终于找到了姜太初留下坐标的这道屏障——但他找不到进去的路。

神秘人在黑暗中站了很久，然后他坐下来，背靠着屏障。没有说话。

陈默在他旁边不远处坐下来。春末的夜风从开阔的山野间吹过来，带着青草和泥土的气息。头顶的星星出来了，在没有灯光的荒野中格外明亮。

陈默靠着屏障，有一点凉意从背后渗进来。他在黑暗中睁着眼睛，没有睡意。

然后他感受到了一道从未有过的信号。

不是以往的"想"——不是想看某个东西那种安静的念头，不是想触碰某种事物的温和愿望。是一种更清晰、更坚定、像是某种不容忽视的指引的——方向。不是向左，不是向右——是向下。

他的目光落在地面上。

在他坐着的位置旁边，屏障与地面的交界处——那里有一道极细的、几乎看不见的线。不是屏障本身的纹路。是一道笔直的、在屏障底部与地面相接的位置延伸出去的缝隙。

像是一扇门的下沿。

体内那道意识在他注意到那道细线的时候，发出了一道极其微弱的、像是确认的亮光。

——这里。

它说。不是用语言，但陈默清晰地接收到了那个意思。

——这里。

陈默低下头，用手指沿着那道缝隙摸了一下。不是岩石的裂缝，不是风化的裂痕——是一道人工的、笔直的、在黑暗中几乎不可能被发现的接缝。他的手指沿着它摸到了尽头——在左侧大约一臂宽的位置，缝隙拐了一个直角，向上。

是一扇门。一扇和屏障表面完全融为一体、没有任何把手、没有任何可见标记的门。

陈默抬起头，看向神秘人坐着的位置。

神秘人在黑暗中睁着眼睛——他也看到了。陈默的手指沿着那道缝隙摸过去的时候，他的目光一直在跟着那根手指移动。

两个人沉默着。在黑暗中，在一扇找到了的门前。体内的灯安静地亮着——像是一个引路的人，在把人带到了目的地之后，安静地退到了一旁。

【第117章·完】
