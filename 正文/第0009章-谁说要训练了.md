# 第9章 谁说要训练了

三月十五，清晨。

陈默是被系统的声音吵醒的。

【宿主，距离与柳鸣的约战还有两天。系统已生成三套训练方案。请宿主选择一套进行今日训练。】

陈默把被子拉过头顶：“……我再睡一会儿。”

【宿主，训练方案已生成。若今日不开始训练，剩余时间将不足以完成最低限度的战斗准备。】

“大不了到时候认输。”

【根据系统对宿主性格的分析——宿主不会认输。】

陈默掀开被子：“你这么了解我？”

【系统在绑定期间对宿主的行为模式进行了持续分析。结论：宿主在社交场合中，宁可硬撑也不愿当众示弱。因此——宿主不会认输。】

“……你分析我干什么？”

【系统通过分析宿主行为模式来优化辅助策略。这是系统功能的正常范畴。】

陈默无话可说。他坐起来揉了揉眼睛。

“行吧，方案呢？”

光幕展开，上面列着三套方案：

─────────────────────
方案A（推荐）：全天高强度训练
- 早课：基础体术+灵力控制（2小时）
- 午课：火法实战训练（2小时）
- 晚课：对抗模拟+战术复盘（2小时）
- 预计效果：胜率提升至8%

方案B（温和）：分段训练
- 早课：基础体术（1小时）
- 午课：火法练习（1小时）
- 晚课：自由练习（1小时）
- 预计效果：胜率提升至6%

方案C（摸鱼）：最低限度
- 训练内容：仅保留火法练习（1小时）
- 其余时间由宿主自行安排
- 预计效果：胜率提升至5%
─────────────────────

陈默毫不犹豫：“C。”

【系统已预判宿主的选择。】

“那你干嘛还要我做选择题？”

【系统认为——提供选择权可以增强宿主的参与感。虽然宿主每次都选C，但选择的过程本身具有心理价值。】

“……你还研究心理学？”

【系统数据库包含基础心理学模型。主要用于预测宿主的行为决策。】

陈默一边穿衣服一边嘟囔：“你这系统越来越不对劲了。”

【系统运行一切正常。】

陈默洗漱完毕，揣着火石和功法出了门。

清晨的外门很安静，大部分人还在睡觉。

他找了块没人的空地——在后山边缘的一块平地上，周围都是树，隐蔽又安静。

“行吧，练什么？”

【请宿主先进行基础体术训练：俯卧撑五十个、深蹲五十个、折返跑十趟。】

“……这是修炼还是军训？”

【基础体术训练可以改善宿主的身体素质，提升战斗中的反应速度和耐力。这是任何战斗训练的前提。】

陈默叹了口气，趴下去开始做俯卧撑。

做到第十五个的时候，手臂开始发抖。

做到第二十五个的时候，他趴在地上不动了。

【宿主，距离目标还差二十五个。】

“休息，我需要休息。”

【宿主已经休息了十秒。】

“不够。”

【系统建议宿主不要——】

“要不这样，”陈默坐起来喘着气，“我做二十五个俯卧撑、二十五个深蹲、五趟折返跑——然后直接进入火法练习。这样效率更高。”

【系统评估：此方案的实际训练效果约为完整方案的百分之六十。但考虑到宿主当前的身体素质——百分之六十也属于可接受范围。】

“那就这么定了。”

陈默咬着牙完成了缩水版的体术训练。等他终于站到空地上准备练火法的时候，已经过去了大半个小时。

他擦了把汗，掏出火石。

“基础火法……怎么用来战斗？”

【《基础火法·入门篇》包含三种攻击方式：火球术、火焰喷射和火盾防御。其中火球术是练气期最常用的攻击手段。请宿主尝试凝聚火球——瞄准前方那棵树。】

陈默握紧火石，调动灵力。

这一次，他的动作比昨天熟练了不少——火球在掌心凝聚，虽然还是摇摇晃晃的，但至少稳定了。

他瞄准十步外的一棵老槐树，把火球推了出去。

火球歪歪扭扭地飞出去，撞在树干上，噗地灭了。

树干上留下一个巴掌大的焦痕。

“……就这？”

【火球术的威力与宿主的灵力输出强度、控制精度和熟练度相关。以宿主练气三层的修为，目前的威力属于正常水平。】

“那这打到柳鸣身上，不是跟挠痒痒一样？”

【如果直接命中要害部位——仍可造成一定伤害。考虑到宿主的修为差距，建议宿主不要依赖正面火力压制，而是采用更灵活的战术。】

“比如？”

【利用火石的瞬时激发特性，制造干扰或烟雾，为近身攻击创造机会。】

陈默愣了一下：“火石还能这么用？”

【低级火石经灵力激发后可瞬发火焰弹，威力约为练气二层修士全力一击。虽然伤害不高，但产生的火光和烟雾足以在短暂时间内干扰对手的视线。】

陈默掏出一块火石，握在手心，试着往里面注入灵力。

火石的表面开始发烫，然后——噗的一声，一道明亮的火焰从指缝间喷出来，伴随着一股灰白色的浓烟。

他被呛得直咳嗽。

“……这也太猛了吧？”

【低级火石的瞬时激发模式，适合用于短时间内的干扰和牵制。建议宿主在战斗中配合火球术交替使用——先以火石烟雾干扰对手视线，再以火球术实施攻击。】

陈默看着手里那块已经熄灭的火石——表面已经裂开了，显然是一次性用品。

“一块火石只能用一次？”

【正确。低级火石为消耗品。但宿主每日签到可获得三块——也就是说，每天有三发烟雾弹可用。】

陈默站在清晨的树林里，手里攥着那块报废的火石，脑海里开始构建一个模糊的画面——

先扔火石制造烟雾 → 趁柳鸣视线受阻时侧移 → 从侧面打出一发火球。

虽然这个战术还很粗糙，但至少——不是完全没有胜算了。

【系统检测到宿主正在进行战术构思。这是积极的转变。系统建议宿主进行至少三十次火球术练习和十次火石烟雾投放练习，将战术动作转化为肌肉记忆。】

“……三十次？”

【鉴于宿主选择了最低限度训练方案，系统已将建议练习量降至最低。标准训练量为一百次。】

陈默深吸一口气，重新握紧一块新的火石。

“行，三十次就三十次。练吧。”

【系统记录：宿主首次无异议接受训练建议。】

“你要是敢把这句话上传给那个什么委员会——”

【系统已将记录标记为“仅供本地参考”。】

陈默愣了一下，然后笑了。

他抬手，对准那棵老槐树，又打出一发火球。

这一次，火球的飞行轨迹比刚才直了一些。

【第9章·完】
